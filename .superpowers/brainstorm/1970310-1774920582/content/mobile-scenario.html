手机端使用场景

你主要在什么场景下用手机访问 Phosphor？这会影响我们的布局优先级

A

随时随地 Vibe Coding

手机就是主力终端 —— 通过语音输入 + 快捷指令，在手机上完成大部分 coding 工作。终端是核心，文件浏览和编辑器是辅助。

B

应急远程操控

主要在电脑上工作，手机用于紧急情况 —— 比如服务器出问题时快速 SSH 进去看看日志、重启服务。需要快速、可靠。

C

两者兼顾

既要能日常 vibe coding，也要能应急操控。手机端需要在两种模式间流畅切换。
