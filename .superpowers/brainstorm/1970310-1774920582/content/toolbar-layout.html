底部工具栏按键布局

Vibe Coding 场景下，你需要哪些快捷键？以下是两种方案

A: 精简方案

只保留最高频的按键，减少视觉干扰，终端区域更大

ESC TAB ↑ ↓ ← →
输入命令...
🎤
6 个按键 + 输入框 + 语音按钮

B: 完整方案

覆盖 Vibe Coding 全场景，两行按键，Ctrl 可组合

ESC TAB Ctrl Alt ⏎
↑ ↓ ← → Ctrl+C Ctrl+Z
输入命令...
🎤
11 个按键（两行）+ 输入框 + 语音
Ctrl 为粘性键：点一次激活，再点其他键组合

💡 Ctrl 粘性键设计

点击 Ctrl 后它保持高亮激活状态，再点其他键自动组合发送（如 Ctrl+C、Ctrl+L、Ctrl+D），发送后 Ctrl 自动取消。类似 iOS 键盘的 Shift 键行为。
